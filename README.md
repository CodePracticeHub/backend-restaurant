# IMPORTANT
## BEFORE START WORKING, DO THE FOLLOWING STEPS. 
- ``` git checkout main ```
- ```git pull origin main```
- ```git checkout <branch_name>```
- ```git merge main```
### That will ensure all branches are up to date with the main branch
